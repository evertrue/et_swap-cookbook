# et_swap cookbook

Aggressively disables any kind of swap partition.

# Requirements

Optimized for (but does not require) EC2

# Usage

Either:

include_recipe "et_swap::disable"

or add it to a run list.

# Attributes

None (currently).

# Recipes

`disable` - The only recipe in the cookbook.
